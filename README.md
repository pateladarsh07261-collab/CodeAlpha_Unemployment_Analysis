# 📊 Unemployment Analysis with Python

## 📌 Project Overview

This project analyzes unemployment trends in India using Python. The analysis focuses on understanding unemployment rates across different states, identifying trends over time, and visualizing the impact of unemployment using data analysis techniques.

This project was completed as part of the **CodeAlpha Data Science Internship**.

---

## 🎯 Objectives

- Analyze unemployment data in India.
- Perform data cleaning and preprocessing.
- Identify state-wise unemployment trends.
- Visualize unemployment patterns using charts.
- Understand the impact of unemployment during the COVID-19 period.

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

Dataset contains:

- Region
- Date
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)

---

## 📈 Data Analysis Performed

- Data Cleaning
- Missing Value Check
- Date Formatting
- State-wise Average Unemployment Rate
- Monthly Unemployment Trend
- Top 10 States by Unemployment Rate
- Estimated Employment Analysis

---

## 📊 Visualizations

- 📌 Bar Chart – Top 10 States by Average Unemployment Rate
- 📌 Line Chart – Monthly Unemployment Trend
- 📌 Bar Chart – Top 10 States by Estimated Employment

---

## 📌 Results

- Unemployment increased during the COVID-19 period.
- Some states experienced significantly higher unemployment rates.
- Employment varied across different regions.
- Visualizations clearly show unemployment trends over time.
